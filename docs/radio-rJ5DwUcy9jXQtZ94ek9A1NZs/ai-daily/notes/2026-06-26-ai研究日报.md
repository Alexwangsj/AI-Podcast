## 模型与能力升级

### 1. Qwen-AgentWorld：面向通用 agent 的语言世界模型

来源：

- Qwen-AgentWorld GitHub repository  
  https://github.com/QwenLM/Qwen-AgentWorld
- Qwen-AgentWorld-35B-A3B Hugging Face model card  
  https://huggingface.co/Qwen/Qwen-AgentWorld-35B-A3B
- Qwen blog: Qwen-AgentWorld: Language World Models for General Agents  
  https://qwen.ai/blog?id=qwen-agentworld

关键事实：

- 2026-06-24，Qwen 发布 Qwen-AgentWorld-35B-A3B 和 AgentWorldBench。
- 开源内容包括 Qwen-AgentWorld-35B-A3B 模型权重和 AgentWorldBench 评测。
- 模型定位是 language world model，用长链路 reasoning 模拟 agent 环境。
- 覆盖七个统一域：MCP、Search、Terminal、SWE、Android、Web 和 OS。
- GitHub README 标注模型为 MoE，35B total / 3B active，256K context。
- 训练管线分三阶段：CPT 注入环境知识，SFT 激活 next-state-prediction reasoning，RL 提升 simulation fidelity；训练数据来自 1000 万以上真实世界交互轨迹。
- README 给出的 AgentWorldBench 表格显示，Qwen-AgentWorld-35B-A3B overall 56.39，Qwen-AgentWorld-397B-A17B overall 58.71；397B 版本在该表中略高于 GPT-5.4 的 58.25。该 benchmark 由发布方提供，需要作为 release benchmark 解读。
- 部署支持 SGLang 和 vLLM，可提供 OpenAI-compatible API；README 建议保持至少 128K context，因为多轮环境模拟依赖长上下文。

技术解读：

- 这不是普通 coding model，而是把“环境状态如何随 action 变化”变成训练目标。agent 做长程任务时，真正困难点通常不是生成下一句，而是理解工具调用、网页状态、终端状态、文件系统和外部世界如何响应。
- Qwen-AgentWorld 的意义在于：如果世界模型足够可靠，agent 可以先在模拟环境中练习、对抗、发现弱点，再把 policy 带回真实环境，这会降低真实执行成本和风险。
- 对开发者来说，短期更现实的用法可能不是直接替换通用 LLM，而是做 agent 训练、测试、benchmark、轨迹合成和 failure case mining。

## 研究论文与机器人/具身智能

### 11. Agent-Authored World Modeling：训练目标从 next observation 转为 decision needs

来源：

- arXiv: Beyond Next-Observation Prediction: Agent-Authored World Modeling for Sequential Decision Making  
  https://arxiv.org/abs/2606.25421

关键事实：

- 提交时间：2026-06-24。
- 论文指出，很多 LLM agent world modeling 工作把目标设为 next-observation prediction，但下一个 observation 不一定包含当前决策真正需要的环境动态。
- 提出 Agent-Authored World Modeling：在每个 state，agent 先识别行动前需要理解什么，再检索相关 transition evidence，并合成训练目标。
- 目标是让 world model 学 decision-oriented dynamics，而不是机械重构下一个 observation。

技术解读：

- 这和 Qwen-AgentWorld 形成很好的理论互补。Qwen 是产业级发布，AAWM 是训练目标设计上的研究信号。
- 对 agent 训练来说，关键问题正在变成：我们到底希望 world model 预测什么。预测网页下一屏、终端下一行固然重要，但更重要的是预测“哪些环境规律会影响下一步决策”。

### 12. BrainAgent：面向脑信号理解的 LLM 多智能体框架

来源：

- arXiv: BrainAgent: A Large Language Model-Driven Multi-Agent Framework for Autonomous Brain Signal Understanding  
  https://arxiv.org/abs/2606.25400

关键事实：

- 提交时间：2026-06-24。
- 论文提出 BrainAgent，用 LLM-driven multi-agent framework 把自然语言意图落到可执行、端到端的脑信号处理 pipeline。
- 系统采用 central supervisor 协调 specialized sub-agents，支持任务分解、执行和 benchmark 评估。
- 目标是降低脑机接口和脑信号分析的专业门槛，并支持更长程、更复杂的分析 workflow。

技术解读：

- BrainAgent 的价值不是通用聊天，而是把专家工作流编码成多 agent 管线。它代表 agent 研究从软件工程、浏览、搜索，进入生物医学信号等高门槛领域。
- 风险也很明显：这类系统必须具备可追溯的数据处理、可解释中间结果和专家复核机制，否则自动化会放大错误。

### 13. General Intuition 与具身世界模型：游戏行为数据作为物理直觉训练源

来源：

- TechCrunch: General Intuition's $2.3B bet that video games can train AI agents for the real world  
  https://techcrunch.com/2026/06/25/general-intuitions-2-3b-bet-that-video-games-can-train-ai-agents-for-the-real-world/

关键事实：

- TechCrunch 2026-06-25 报道，General Intuition 获得 3.2 亿美元融资，估值 23 亿美元。
- 公司使用大量游戏视频和 action data 训练具备 spatial-temporal reasoning 的模型。
- 报道称其 world model 内部被称作 training environment 或 gym，最终目标是出售 agentic model。
- 资金将主要用于扩展 compute capacity，并计划在夏末前扩大 API 可用性。

技术解读：

- 这条路线与机器人真实数据相比成本更低、规模更大，但 sim-to-real gap 仍然是关键问题。短期可能更适合训练通用空间-时间表征、规划和交互直觉，再与真实机器人数据做对齐。
